# Kvalitet Zraka - Kakanj

Ovaj projekat prikazuje **trenutni kvalitet zraka u gradu Kakanj** koristeći podatke sa [WAQI API](https://aqicn.org/api/). Stranica je izrađena koristeći **HTML, CSS i Vanilla JavaScript** te **Bootstrap** za responzivni dizajn.

**Pristup preko GitHub Pages:** [https://eldansehagic.github.io/air-quality/](https://eldansehagic.github.io/air-quality/)

---

## Šta stranica prikazuje

- **Index kvaliteta zraka (AQI):** trenutna vrijednost AQI indeksa sa vizuelnom oznakom boja  
- **Dominantni zagađivač:** trenutno najviše prisutni polutant  
- **Poruka:** kratke i jasne informacije vezane za kvalitet zraka  
- **Zadnje ažuriranje:** datum i vrijeme posljednjeg očitavanja podataka  

Stranica koristi tamnu temu za ugodan vizuelni doživljaj, a boje poruka su prilagođene vrijednostima AQI indeksa, olakšavajući brzo razumijevanje trenutnog stanja kvaliteta zraka.

---

## Tehnologije

- **HTML** – struktura stranice  
- **CSS** – stilovi i tamna tema  
- **Bootstrap** – responzivni dizajn i kartice  
- **Vanilla JavaScript** – dohvat i prikaz podataka iz WAQI API-ja  
- **WAQI API** – podaci o kvalitetu zraka
